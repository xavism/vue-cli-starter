> Little tutorial about using Apollo Client and GraphQL in your Vue.js projects.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## Install and configure Apollo Client

> First of all We need to install apollo-client and vue-apollo packages

``` bash
# install packages
npm install —save apollo-client vue-apollo
```




For detailed explanation on how things work, checkout the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
